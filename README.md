### 用户注册流程

- /api/notify/v1/captcha?type=register 图形验证接口 获取图形验证码
- /api/notify/v1/send_code 短信验证接口 获取短信（判断 1 图形验证码未过期且验证码正确 2 短信验证码过期或未发送的情况下才发送
- "/api/user/v1/register 注册接口 通过用户和短信注册

### 返回值（code, msg, data）统一管理工具类

- BackCode 类
  - 统一封装状态成功/失败的 res 格式
- CodeEnum
  - 存储不同错误的状态码和 msg
